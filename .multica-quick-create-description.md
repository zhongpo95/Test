## User request

게임에 부족한 부분이 많아서 해야 할 작업이 많다. 아래 우선 처리해야 할 작업들의 작업 순서를 어떻게 하면 좋을지 정하고, 작업 순서에 맞춰서 새 이슈로 나눠줬으면 한다.

1. 아이템 인벤토리 변경
인벤토리에 포션이 표시되어 있는데, 포션은 무제한으로 변경할 예정이라서 표시가 따로 필요 없을 것 같다. 그에 따른 디자인을 조금 수정했으면 좋겠다.

2. 스텟창 UI 변경
능력치를 보는 스텟창에 장착된 아이템이 표시되어 있는데, 정렬도 제대로 되어 있지 않고 디자인도 워크래프트에 어울리게 바꾸고 싶다.

3. 보스 체력 수정
원래 만들려는 게임에서는 데미지 수치가 높아서 체력 단위를 쪼개는 방식을 사용했었는데, 이제는 필요 없어질 것 같아서 이 부분을 수정하고 싶다.

우선으로 처리해야 될 이슈는 이 세 가지 정도다.
