# UI_Pick texture source mapping

This file records the UI_Pick texture files that remain in the repository after the runtime TGA cleanup.

## Runtime TGA textures

- war3mapImported/UI_Pick_Backdrop.tga is used by the A backdrop frame in UI/UI_Pick.j.
- war3mapImported/UI_Pick_Panel.tga is used by the I preview panel and J skin panel frames in UI/UI_Pick.j.
- war3mapImported/UI_Pick_Frame.tga is used by the card-list frame in UI/UI_Pick.j.
- war3mapImported/UI_Pick_NameBar.tga is used by each hero-name bar in UI/UI_Pick.j.

## Retained source PNG attachments

Large source TGA files and older source TGA attachments were removed from war3mapImported because UI_Pick no longer references them at runtime.
